*Install
If you are using URP,please import the package in "/Chracter Boom/URP Patch/"

*How to use Chracter Boom?
1.Please view /Chracter Boom/Scenes/Tutorials

*How to use Sprite Blend Shader?
1.Click "add Component" in inspector,add "SetMatColor" component to the GameObject.
2.Set material In the inspector of SetMatColor .
3.Custom your color In the inspector of SetMatColor.

*How to set _SecondTex and _ThirdTex?
1.Open the SpriteEditor.
2.Select "Secondary Textures".
3.Set your _SecondTex and _ThirdTex in the SpriteEditor.
